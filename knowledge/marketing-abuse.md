# 营销活动滥用 / Marketing Abuse

## 适用场景

新人券、优惠券、红包、积分、赠品、拉新活动、返利活动、大促活动、会员权益等被规模化套利。

## 常见 BREAK 风险

- `R0009` 恶意薅羊毛
- `R0005` 营销活动作弊
- `R0054-003` 闪退套利
- `R0002` 优惠券枚举
- `R0034` 自动化养号
- `R0099` 黑 IP 识别绕过

## 常见链路

```text
手机号/接码资源 → 批量注册账号 → 代理 IP / 云手机 / 模拟器 → 领取权益 → 集中下单或核销 → 退款 / 转卖 / 补贴套利
```

## 观测信号

- 新号注册时间集中
- 注册到领券/下单路径过短
- 同批权益集中领取和核销
- IP/ASN、设备、地址、支付工具聚集
- SKU、券批次、下单时间高度集中
- 高退款率或权益获取后退款

## 调查要点

- 账号-设备-IP-手机号-支付工具-地址图谱
- 领券、下单、退款之间的时间间隔
- 活动权益 ROI 和正常用户基线偏差
- 同一设备族或 IP 池下的多账号行为

## 治理建议

- 新号权益风险评分
- 权益延迟发放或分层发放
- 高频领券/核销频控
- 高风险订单延迟发货或复核
- 退款后权益回收和活动补贴保护
