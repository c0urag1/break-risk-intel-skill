# 自动化养号 / Account Farming

## Input

```text
我们发现一批账号每天固定时间自动浏览商品、点赞、评论、收藏。
这些账号行为看起来像正常用户，但内容模板很相似。
一段时间后，这些账号开始参与抢购和领券活动。
帮我判断这是什么风险。
```

## Expected Output Shape

### 风险判断

自动化养号。账号先通过低风险行为提升可信度，再被用于抢购、领券、刷量或其他套利场景。

### BREAK 映射

- `R0034` 自动化养号
- `R0001` 流程自动化
- `R0088` 自动化登录风险
- `R0003` 恶意抢购
- `R0009` 恶意薅羊毛

### 黑灰产链路假设

```text
批量注册账号 → 自动登录 → 浏览 / 点赞 / 评论 / 收藏 → 降低账号风险评分 → 参与抢购 / 领券 / 刷量 → 转卖权益或套利
```

### 观测信号

- 多账号行为时间窗口一致
- 评论、点赞、收藏节奏相似
- 评论文本模板相似
- 账号之间设备/IP/UA 特征重合
- 低价值行为持续一段时间后转向高价值活动
- 行为路径机械、缺少正常用户随机性

### 建议动作

- 建立账号生命周期风险评分
- 对低价值行为和高价值行为做联动分析
- 检测账号行为模板相似度
- 建立账号-设备-IP-行为序列聚类
- 对“养号后参与活动”的账号单独建模
