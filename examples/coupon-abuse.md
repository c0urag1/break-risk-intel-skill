# 新人券批量套利

案例类型：模拟案例。
来源：分析框架总结 + BREAK 风险 ID。

## 输入

```text
大促期间，大量新号通过代理 IP 注册，领取新人券后集中下单低价商品。
这些账号注册到下单间隔很短，部分订单随后退款。
帮我分析可能的黑灰产链路，并输出风险矩阵和调查计划。
```

## 已知事实

- 大促期间新号数量异常增加。
- 新号通过代理 IP 注册。
- 新号领取新人券后快速下单低价商品。
- 部分订单后续退款。

## 合理假设

| 假设 | 置信度 | 依据 | 证据缺口 |
|---|---|---|---|
| 新人权益被批量套利 | 高 | 新号、代理 IP、领券、低价 SKU、退款集中出现 | 需要券核销、订单、退款、账号图谱验证 |
| 账号池或接码资源参与 | 中 | 大量新号集中注册 | 需要手机号段、注册设备、IP、验证码行为验证 |
| 存在退款套利或转卖变现 | 中 | 部分订单退款 | 需要退款理由、退款时间、商品流向、权益回收情况 |

## 攻击标的

- 新用户补贴和新人券资格。
- 低价 SKU 与券面额之间的套利空间。
- 退款规则和权益回收规则。
- 大促期间风控策略放宽或审核延迟窗口。

## 角色分工

| 角色 | 状态 | 置信度 | 依据 | 证据缺口 |
|---|---|---|---|---|
| 组织者/中介 | 推断 | 中 | 批量新号和一致路径 | 需要跨账号组织关系、收货/支付/设备聚类 |
| 账号资源方 | 推断 | 中 | 大量新号 | 需要手机号、设备、注册时间分布 |
| 代理/IP 资源方 | 已观察 | 高 | 用户输入提到代理 IP | 需要 ASN、IP 段、代理标签 |
| 执行者 | 推断 | 中 | 领券后集中下单 | 需要行为序列和订单路径 |
| 套现/转卖方 | 可能 | 低 | 退款或低价商品套利可能 | 需要退款、转卖、资金流证据 |

## 资源偏好

| 资源 | 状态 | 依据 | 待验证数据 |
|---|---|---|---|
| Account pool | 较可能 | 大量新号集中注册 | 注册时间、资料模板、生命周期 |
| Phone / SMS resources | 可能 | 新号批量注册通常需要验证资源 | 手机号段、虚拟号、绑定变更 |
| Proxy IP | 已观察 | 用户输入明确提到代理 IP | IP 风险标签、ASN、地理分布 |
| 设备环境 | 可能 | 批量账号通常需要设备隔离 | 设备指纹、模拟器/云手机信号 |
| Address / payment tools | 可能 | 下单和退款链路可能复用 | 收货地址、支付工具、退款账户聚类 |

## 概念级作弊链路

```text
权益目标选择 → 账号/网络资源准备 → 批量注册 → 领取新人券 → 低价 SKU 下单 → 退款/转卖/补贴套利 → 账号废弃或复用
```

这是防护侧的概念模型，不是执行指南。

## 变现路径

- 通过优惠券抵扣获取商品价差。
- 通过退款规则保留部分权益或套利空间。
- 通过转卖商品、权益或账号实现间接变现。

## BREAK 映射

- `R0009` 恶意薅羊毛
- `R0005` 营销活动作弊
- `R0034` 自动化养号
- `R0099` 黑 IP 识别绕过
- `R0054-003` 闪退套利，若证据显示退款规则被用于套利

## 观测信号

- 账号： 注册时间集中、资料模板、账号生命周期短。
- 设备/IP： 代理 IP、ASN 聚集、设备指纹相似。
- 优惠券： 领取路径一致、券核销率异常、同批券集中使用。
- 订单： 注册到下单间隔短、SKU 集中、客单价异常。
- 退款： 退款率、退款理由、退款时间窗口。
- 图谱： 账号-设备-IP-手机号-地址-支付工具-退款账户关联。

## 风控切点

- 新号领券风险评分。
- 高风险账号延迟发券或降低券面额。
- 注册到用券增加冷却期或分层校验。
- 异常订单延迟发货或人工复核。
- 退款后权益回收和补贴保护。
- 图谱聚类识别账号池和资源复用。

## 误伤风险

- 大促自然增长可能导致新号和低价 SKU 集中。
- 新用户真实购买也可能注册后快速下单。
- 不应只凭代理 IP 或低价 SKU 判定，应结合账号、设备、券、订单、退款和图谱证据。
