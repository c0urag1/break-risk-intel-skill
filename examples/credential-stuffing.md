# 撞库与账号接管 / Credential Stuffing & Account Takeover

## Input

```text
最近登录接口失败率明显升高。
很多账号在短时间内从不同 IP 尝试登录。
部分成功登录后立即修改手机号、绑定新设备、使用账户余额。
帮我判断这是不是撞库或账号接管。
```

## Expected Output Shape

### 风险判断

撞库 / 账号接管风险。异常链路包括批量登录尝试、成功登录后的敏感信息修改，以及账户资产使用。

### BREAK 映射

- `R0032` 撞库
- `R0084` 账号盗用
- `R0090` 账号接管
- `R0099` 黑 IP 识别绕过
- `R0049` 代登录、代下单

### 黑灰产链路假设

```text
泄露账号密码库 → 批量登录验证 → 命中有效账号 → 修改绑定信息 → 使用余额 / 积分 / 优惠券 → 转卖账号或资产变现
```

### 观测信号

- 登录失败率异常升高
- 同 IP 多账号尝试
- 同账号多 IP 尝试
- 登录成功后立即修改手机号或密码
- 新设备绑定后立即发生交易
- 账户余额、积分、券被快速消耗
- 登录地与历史常用地差异明显

### 建议动作

- 对异常登录增加二次验证
- 对登录成功后的敏感操作增加冷却期
- 对密码撞库行为做 IP、设备、账号三维频控
- 对高风险登录会话限制资产操作
- 对疑似被接管账号触发用户通知和冻结保护
