# 账号风险 / Account Risk

## 适用场景

批量注册、自动化养号、撞库、账号接管、异常登录、代登录、账号权益倒卖。

## 常见 BREAK 风险

- `R0034` 自动化养号
- `R0001` 流程自动化
- `R0032` 撞库
- `R0032-001` 撞库(凭证填充)
- `R0084` 账号盗用
- `R0090` 账号接管
- `R0088` 自动化登录风险
- `R0049` 代登录、代下单

## 常见链路

```text
账号资源 / 泄露凭证 → 批量登录或注册 → 养号或验证可用性 → 修改绑定信息 / 参与活动 / 使用资产 → 转卖或套利
```

## 观测信号

- 登录失败率异常升高
- 同 IP 多账号尝试、同账号多 IP 尝试
- 新设备绑定后立即发生敏感操作
- 登录地、设备、行为序列与历史基线偏离
- 批量低价值行为后转向高价值活动
- 评论、点赞、浏览、收藏节奏高度相似

## 调查要点

- 登录失败与成功事件的时间窗口
- 账号-设备-IP-手机号绑定变化
- 敏感操作前后的会话风险
- 是否存在账号池或养号行为

## 治理建议

- 异常登录二次验证
- 敏感操作冷却期
- 账号生命周期风险评分
- 账号-设备-IP 图谱聚类
- 高风险会话限制资产操作
