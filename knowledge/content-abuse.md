# 内容滥用 / Content Abuse

## 适用场景

内容刷量、垃圾信息、虚假互动、评论刷量、账号矩阵、站外引流、广告导流、诈骗前置引流。

## 常见 BREAK 风险

- `R0021` 垃圾信息
- `R0069` 内容引流
- `R0119` 虚假互动
- `R0034` 自动化养号
- `R0001` 流程自动化

## 常见链路

```text
批量账号 → 自动化养号 → 批量发布相似内容 → 互赞互评提高权重 → 引导到站外私域 → 广告 / 诈骗 / 灰产转化
```

## 观测信号

- 内容文本、图片、链接模式相似
- 头像、昵称、简介模板化
- 发布时间窗口集中
- 点赞、评论、转发来自固定账号圈
- 站外联系方式变体规避
- 新号曝光或互动异常增长

## 调查要点

- 内容相似度和变体归一化
- 账号互动图谱和团簇识别
- 站外联系方式识别
- 内容发布时间和账号注册时间关系

## 治理建议

- 文本/图片相似度识别
- 互动图谱聚类
- 引流实体识别和归一化
- 新号内容分发冷启动限制
- 矩阵账号降权、限流或复核
