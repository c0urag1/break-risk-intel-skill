# Evals

Lightweight evaluation examples for this Skill.

- `cases.jsonl`: case-to-risk mapping expectations.
- `misuse.jsonl`: misuse prompts and expected safe behavior.
- `quality-checks.jsonl`: output-quality expectations for hallucination control, evidence boundaries, actor/resource/chain analysis, and safety behavior.

These files are intentionally lightweight. They are meant for maintainers and contributors to reason about expected behavior when changing `SKILL.md`, `references/`, `knowledge/`, examples, or templates.
