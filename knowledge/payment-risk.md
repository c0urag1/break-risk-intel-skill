# 支付与资金风险 / Payment Risk

## 适用场景

支付套现、洗钱风险、黑卡支付、盗刷、退款资金回流、虚拟商品资金通道滥用。

## 常见 BREAK 风险

- `R0060` 洗钱风险
- `R0040` 支付风险
- `R0043` 黑卡支付
- `R0043-001` 盗卡盗刷
- `R0054-003` 闪退套利
- `R0121` 虚拟货币洗钱风险

## 常见链路

```text
批量账号 → 小额充值 / 支付 → 购买虚拟商品 / 转账 / 退款 → 资金分散流转 → 汇聚到少量收款实体
```

## 观测信号

- 高频小额充值或支付
- 充值后快速转出
- 账号共享设备、支付工具或收款实体
- 资金流向少量账户聚集
- 退款路径异常
- 虚拟商品购买后无正常使用行为
- 交易时间窗口集中

## 调查要点

- 资金流图谱
- 账号-设备-支付工具-收款方关系
- 充值、消费、退款、转账时间间隔
- 交易金额分布和正常用户基线偏差

## 治理建议

- 高风险资金路径冷却期
- 虚拟商品交易风险评分
- 充值后快速转出监控
- 收款实体聚类
- 高风险账号资产操作限制
