# 内容刷量与引流 / Content Spam & Off-platform Traffic

## Input

```text
某些账号短时间内发布大量相似内容，并互相点赞、评论、转发。
部分内容引导用户添加站外联系方式。
这些账号头像、昵称、发布时间都很相似。
请分析可能的黑灰产风险。
```

## Expected Output Shape

### 风险判断

内容平台常见的刷量 + 引流 + 账号矩阵风险。目标可能是提升内容权重、导流到站外、广告变现或诈骗链路前置。

### BREAK 映射

- `R0021` 垃圾信息
- `R0069` 内容引流
- `R0034` 自动化养号
- `R0001` 流程自动化
- `R0119` 虚假互动

### 黑灰产链路假设

```text
批量账号 → 自动化养号 → 批量发布相似内容 → 互赞互评提高权重 → 引导到站外私域 → 广告 / 诈骗 / 灰产转化
```

### 观测信号

- 内容文本相似度高
- 账号头像、昵称、简介模板化
- 发布时间高度集中
- 点赞/评论/转发来自固定账号圈
- 站外联系方式变体规避
- 账号之间存在明显互动团簇
- 新号内容曝光异常增长

### 建议动作

- 文本相似度和变体识别
- 账号互动图谱聚类
- 站外联系方式识别
- 对新号内容分发增加冷启动限制
- 对疑似矩阵账号做降权、限流或复核
