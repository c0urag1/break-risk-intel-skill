# 支付套现与洗钱风险 / Payment Cash-out & Money Laundering Risk

## Input

```text
我们发现部分账号频繁进行小额充值，然后通过退款、转账、虚拟商品购买等方式快速转出。
这些账号之间存在支付工具和设备关联。
请帮我做风险分析。
```

## Expected Output Shape

### 风险判断

潜在支付套现 / 洗钱 / 资金通道滥用风险。需要结合支付、退款、账户关系、设备关系和资金流转路径进一步验证。

### BREAK 映射

- `R0060` 洗钱风险
- `R0040` 支付风险
- `R0054-003` 闪退套利
- `R0084` 账号盗用
- `R0034` 自动化养号

### 黑灰产链路假设

```text
批量账号 → 小额充值 → 购买虚拟商品 / 转账 / 退款 → 资金分散流转 → 汇聚到少量收款实体
```

### 观测信号

- 高频小额充值
- 充值后快速转出
- 账号之间共享设备或支付工具
- 资金流向少量收款方
- 退款路径异常
- 虚拟商品购买后无正常使用行为
- 交易时间窗口集中

### 建议动作

- 建立资金流图谱
- 监控充值后快速转出
- 对虚拟商品交易增加风险评分
- 对高风险资金路径设置冷却期
- 对账号、设备、支付工具、收款方做关联分析
