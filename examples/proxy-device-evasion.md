# 代理 IP 与设备对抗

案例类型：模拟案例。
来源：分析框架总结 + BREAK 风险 ID。

## 输入

```text
近期注册和下单请求来自大量不同 IP，但行为路径高度一致。
设备指纹有碰撞和异常缺失，一些账号疑似来自模拟器或云手机环境。
帮我分析代理 IP 和设备对抗风险。
```

## 已知事实

- 请求来自大量不同 IP。
- 行为路径高度一致。
- 设备指纹存在碰撞和异常缺失。
- 部分账号疑似来自模拟器或云手机环境。

## 合理假设

| 假设 | 置信度 | 依据 | 证据缺口 |
|---|---|---|---|
| 存在代理/IP 轮换 | 高 | 多 IP 但行为一致 | 需要 ASN、代理标签、IP 切换时序 |
| 存在设备环境伪装或批量设备资源 | 中 | 指纹碰撞、缺失、模拟器/云手机信号 | 需要设备指纹明细和环境检测结果 |
| 设备/IP 对抗服务支撑其他作弊链路 | 中 | 注册和下单路径一致 | 需要关联到领券、下单、退款、刷量等收益链路 |

## 攻击标的

- 注册、登录、下单等链路的频控规则。
- IP/设备维度的风险识别。
- 账号关联图谱。

## 角色分工

| 角色 | 状态 | 置信度 | 依据 | 证据缺口 |
|---|---|---|---|---|
| 代理/IP 资源方 | 推断 | 高 | 大量不同 IP、行为一致 | 需要 IP 风险标签和 ASN 分布 |
| 设备资源方 | 推断 | 中 | 指纹异常和模拟器/云手机迹象 | 需要设备环境验证 |
| 自动化/工具方 | 可能 | 中 | 行为路径高度一致 | 需要请求节奏和客户端路径 |
| Downstream abuse executor | 可能 | 低 | 注册/下单请求 | 需要具体套利或交易收益证据 |

## 资源偏好

- 代理 IP：用于分散请求来源。
- 云手机/模拟器/设备环境：用于多账号隔离。
- 自动化工具：用于重复执行路径。
- 账号资源：用于注册、下单或其他业务行为。

## 概念级作弊链路

```text
网络/设备资源准备 → 账号注册或登录 → 行为路径执行 → 规避 IP/设备关联 → 进入下游业务链路 → 权益/交易/刷量变现
```

## 变现路径

- 本身通常不是最终变现环节。
- 常作为权益套利、刷量、刷单、撞库、爬虫等链路的基础设施。

## BREAK 映射

- `R0099` 黑 IP 识别绕过
- `R0098` 设备指纹识别绕过
- `R0034` 自动化养号，若存在批量账号养成
- `R0088` 自动化登录风险，若出现在登录链路

## 观测信号

- IP/ASN： 代理标签、ASN 聚集、地理跳变、IP 切换频率。
- 设备： 指纹碰撞、字段缺失、模拟器/云手机、传感器异常。
- 行为： 不同 IP/设备下路径高度一致。
- 账号： 同批账号共享设备族、IP 族、行为序列。
- 图谱： 账号-设备-IP-订单/内容/券/登录关系。

## 风控切点

- IP、设备、账号联合风险评分。
- 关键链路增加设备可信度校验。
- 对路径一致但来源分散的行为做聚类检测。
- 高风险设备环境降权或触发二次验证。
- 不单独依赖 IP 或设备字段，采用多信号图谱。

## 误伤风险

- 企业网络、校园网、运营商 NAT、隐私保护工具可能造成 IP 或设备异常。
- 设备指纹缺失可能来自客户端版本或权限限制。
- 应结合行为一致性和业务收益链路判断。
