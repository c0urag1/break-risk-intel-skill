# 设备与网络对抗 / Device & Network Risk

## 适用场景

代理 IP、秒拨、住宅代理、黑 IP 识别绕过、模拟器、云手机、群控、设备指纹伪造、无头浏览器。

## 常见 BREAK 风险

- `R0099` 黑 IP 识别绕过
- `R0050` 风险设备识别绕过
- `R0001-002` 自动化模拟器
- `R0088` 自动化登录风险
- `R0001` 流程自动化

## 常见链路

```text
代理 IP 池 → 云手机 / 模拟器 / 群控 → 批量账号登录 → 自动化领券 / 抢购 / 内容互动 → 权益套利或商品转卖
```

## 观测信号

- IP 分散但设备指纹相似
- 设备参数组合异常
- 传感器数据缺失
- UA、分辨率、系统版本模板化
- 单设备或设备族关联多个账号
- 请求集中命中高价值接口
- 登录、领券、抢购路径过短

## 调查要点

- 账号-设备-IP 三元组聚类
- 设备族识别
- IP/ASN 和地理位置异常
- 高价值接口访问序列
- 设备环境与历史行为基线偏差

## 治理建议

- 设备族识别
- IP 风险和设备风险联合建模
- 高价值接口行为序列校验
- 模拟器/云手机请求降权或复核
- 账号-设备-IP 聚类监控
