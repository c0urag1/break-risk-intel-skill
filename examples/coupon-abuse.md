# 新人券批量套利 / Coupon Abuse

## Input

```text
大促期间，大量新号通过代理 IP 注册，领取新人券后集中下单低价商品。
这些账号注册到下单间隔很短，部分订单随后退款。
帮我分析可能的黑灰产链路，并输出风险矩阵和调查计划。
```

## Expected Output Shape

### 风险判断

营销活动套利 / 新人券批量薅羊毛。

### BREAK 映射

- `R0009` 恶意薅羊毛
- `R0005` 营销活动作弊
- `R0034` 自动化养号
- `R0099` 黑 IP 识别绕过
- `R0054-003` 闪退套利

### 黑灰产链路假设

```text
手机号/接码资源 → 批量注册账号 → 代理 IP / 云手机 / 模拟器 → 自动领券 → 集中下单 → 退款 / 转卖 / 补贴套利
```

### 观测信号

- 注册时间高度集中
- 同 IP 段或同 ASN 聚集
- 设备指纹相似
- 注册 → 领券 → 下单路径过短
- SKU 高度集中
- 退款率异常
- 多账号共享支付工具、地址或设备族

### 建议动作

- 对新号领券增加风险评分
- 对高风险新号延迟发券或降低券面额
- 对账号-设备-IP-地址-支付工具做图谱聚类
- 对异常订单延迟发货或人工复核
- 对退款套利链路增加策略监控
